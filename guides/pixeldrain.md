# How to remove your course from Pixeldrain

> Canonical version: [https://delister.co/remove/pixeldrain](https://delister.co/remove/pixeldrain). This repo mirrors it for reference and machine-readability.

Pixeldrain (pixeldrain.com) is a file host, not a listing site. When your course appears on a piracy aggregator, the download button often points at a Pixeldrain file. Reaching Pixeldrain directly removes the download itself, so every listing pointing at that file goes dead at once.

## Does Pixeldrain honor copyright notices?

Pixeldrain publishes an abuse contact and acts on copyright complaints sent to it. Because it hosts the file rather than merely linking to it, a notice here is higher leverage than one aimed at the listing page. Keep the complaint plain and factual; a clean, specific notice from a reachable sender is what gets acted on.

## The steps

### 1. Find the exact Pixeldrain file URL

Follow the download button on the listing until you land on a pixeldrain.com address, usually of the form `pixeldrain.com/u/` followed by a file ID. Copy that full URL. If one course is split across several Pixeldrain files, collect each one.

### 2. Send a copyright notice to abuse@pixeldrain.com

Email `abuse@pixeldrain.com` and include:

- Your full legal name and country as the rights holder.
- Your course name and a one-line description of the work.
- Your official sales page URL as the authorized example of the work.
- The exact infringing pixeldrain.com file URL or URLs.
- The two sworn statements a copyright notice requires: a good-faith belief that the use is not authorized, and that the information is accurate and you are the rights holder or authorized to act.

Send it from an address on your own domain where a reply can reach you.

### 3. Confirm the removal

Re-open the file URL after Pixeldrain acts. A removed file returns a not-found or removed page instead of the download. Confirm at the file URL, not the listing page.

## A note on scope

Pixeldrain often holds the large, complete archive of a course, which makes it the file worth prioritizing. A listing may show download counts in the hundreds against a single Pixeldrain file; removing that file is what stops the ongoing distribution.
